FREQUENTLY ASKED QUESTIONS

> WHAT IS HANDY?
Handy is a professional-grade Game Builder Tool (Editor Utility) designed for the Fab Marketplace. It streamlines the development process by generating and modifying Data Assets, Blueprints, and Level data without injecting code into runtime game classes.
> WHAT IS BLOOD-YIELD?
Blood-Yield is an upcoming dark fantasy RPG built using Unreal Engine 5. It features a unique "Life-Death-Trade" core loop involving Reapers (harvesters) and Sowers (architects), and utilizes a gritty, non-cartoonish visual style.
