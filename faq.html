What is the "Soul-Blight" mechanic?

Soul-Blight is a progressive decay affecting all refined equipment. If left untreated at the Hjarta Forge, items suffer a massive 80% stat penalty. Restoration requires Empty Vials and Refined Essence.

How does the "Symbiotic Economy" work?

Reapers harvest raw materials (Soul Essence/Cold Iron) from the field. Sowers refine these into Bio-Armor. Both roles depend on the P2P Marketplace for survival and profit.

SECTOR // HANDY_TECHINTEL_02
Is Handy part of my game code?

No. Per the Isolation Protocol, all Handy logic stays in Content/Handy/. It is an Editor-only tool and will not increase your final game's build size or affect runtime performance.

What LLMs does Elite support?

Elite supports all major API-driven models (OpenAI, Claude, etc.) via the BYOB (Bring Your Own Brain) routing bridge. You provide the API key; Handy provides the Unreal connectivity.
