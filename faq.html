To ensure Fab Marketplace readiness and project stability, Handy logic (Python scripts and Editor Blueprints) is strictly isolated in Content/Handy/. It interacts with the game only by generating Data Assets or Actor Blueprints, keeping the runtime game code clean.

Equipment in Blood-Yield is subject to Soul-Blight. If untreated, items suffer an 80% stat penalty. You must use Refined Essence at the Hjarta Forge or mirror waygates to restore equipment integrity.

The Maw is the Bastion’s salvage pit. Breaking down equipment yields approximately a 5% return on Cold Iron, making high-end Bio-Armor refinement a high-stakes decision for Sowers.
