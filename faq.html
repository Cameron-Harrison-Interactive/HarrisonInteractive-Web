What is the "Soul-Blight" system?

Soul-Blight is a progressive equipment decay mechanic. If left untreated, equipment suffers an 80% stat penalty. It must be restored using Empty Vials and Refined Essence at the Hjarta forge.

Reaper vs. Sower Roles?

Reapers focus on the harvest of Soul Essence and Cold Iron from the field. Sowers act as architects, refining these raw materials into Bio-Armor and high-end gear via the Bastion waygates.

Is the Marketplace secure?

Yes. Blood-Yield utilizes a secure RMT marketplace with "Verified Badges" for HI Ultimate subscribers, ensuring transparency in all player-to-player Cold Iron trades.

Where is the tool logic stored?

Per professional standards, all Handy logic (Python scripts and Editor Blueprints) is strictly isolated within the Content/Handy/ directory. It never interferes with runtime game classes.

What does "BYOB" mean in Elite?

Bring Your Own Brain. This feature allows you to route your own LLM API keys directly through Handy's Python bridge to automate node generation and graph cleaning.

Can I sell tools I make with Handy?

Handy Ultimate is designed for enterprise scaling. Assets generated by Handy are yours, but the core Handy modular framework remains the intellectual property of HI.
