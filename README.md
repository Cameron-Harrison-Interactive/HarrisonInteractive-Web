## 📡 OVERVIEW
This repository houses the front-end architecture for **HarrisonInteractive.dev**, the official digital pulse hub for **Harrison Interactive LLC**. 

The site operates on a custom "Omni-Network" Master Stylesheet (v3.2), utilizing a strict 2-column terminal layout, responsive Flexbox navigation, and non-blocking CSS scanline animations to achieve a high-performance "Cyber-Tech" aesthetic.

## 📂 DIRECTORY STRUCTURE
The hub is completely static (HTML5/CSS3) for maximum speed and security, organized into the following active modules:

* **`index.html`** - The Master Hub / Broadcast Node.
* **`handy.html`** - Technical specifications for the **HANDY** Editor Utility.
* **`about.html`** - The Studio Manifesto and Core Directives.
* **`faq.html`** - The Signal Terminal for support and general inquiries.
* **`legal.html`** - Unified Privacy Policy and Terms of Service.
* **`styles.css`** - The Omni-Network Engine handling the grid and styling.
* **`Blood-Yield/`** - Sub-directory dedicated to the Dark Fantasy RPG project.

## 🛠️ ACTIVE PROJECTS

### 1. HANDY [ Module: Editor Utility ]
The apex of UE5 automation. A hybrid logic architecture merging visual Blueprint accessibility (`EUW_HI_HandyMain`) with the raw power of Python (`handy_router.py`). Built for industrial-scale project management.
